## Relacionales

Operadores Relacionales

  • Mayor que: > (>).
  • Menor que: < (<).
  • Mayor o igual que: \(\geq\) (>=).
  • Menor o igual que: \(\leq\) (<=).
  • Igualdad: = (==).
  • Desigualdad: \(\neq\).
    C++ Python
    != !=
    <>
## Operadores Relacionales: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int a = 0.5; bool f = (0 < a && a <= 1); cout << a << endl; cout << f << endl; return 0; } ```
## Operadores Relacionales: Ejemplo ```python a = 0.5 f = (0 < a and a <= 1) print(a) print(f) ```
