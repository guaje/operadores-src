## Incremento
## Incremento El operador ++ se puede utilizar como un operador __prefijo__ (++i) o __postfijo__ (i++), y dependiendo de la posición donde se ubique entonces la asignación se hace antes o después del incremento.
## Preincremento __++x__: La variable x se incrementa en 1 y luego se retorna el nuevo valor de la variable.
## Preincremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = ++x; cout << x << endl; cout << y << endl; return 0; } ```
## Preincremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; x += 1; int y = x; cout << x << endl; cout << y << endl; return 0; } ```
## Postincremento __x++__: Retorna el actual valor de la variable x y luego incrementa su valor en 1.
## Postincremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = x++; cout << x << endl; cout << y << endl; return 0; } ```
## Postincremento: Ejemplo ```cpp #include <iostream> using namespace std; int main() { int x = 2; int y = x; x += 1; cout << x << endl; cout << y << endl; return 0; } ```
