## Operador Ternario
## Operador Ternario: C++ ```cpp #include <iostream> using namespace std; int main() { char a = 'A'; bool b = (65 <= a && a <= 90) ? true : false; cout << a << endl; cout << b << endl; return 0; } ```
## Operador Ternario: Python ```python a = 'A' b = True if (65 <= ord(a) and ord(a) <= 90) else False; print(a) print(b) ```
